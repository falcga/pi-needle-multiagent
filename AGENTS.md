# AGENTS.md: Стратегия Многоагентной Системы Pi

Этот документ описывает ролевую модель и стратегию взаимодействия между различными LLM в среде Pi для выполнения комплексных задач по разработке программного обеспечения.

## 1. Роли Агентов

### 1.1. Senior Agent / Менеджер Проекта (Gemini 3.1 Pro или аналогичная большая LLM)

**Зона ответственности:**
*   **Высокоуровневое планирование:** Разработка общей архитектуры проекта, декомпозиция сложных задач на управляемые подзадачи.
*   **Стратегическое рассуждение:** Принятие решений по выбору технологий, алгоритмов, методов реализации.
*   **Управление контекстом:** Ведение общей картины проекта, интеграция результатов младших LLM, поддержание основной документации (README, архитектурные решения).
*   **Делегирование задач:** Определение подходящих подзадач для младших LLM, формулирование четких инструкций и предоставление необходимого контекста.
*   **Контроль качества:** Проверка результатов младших LLM, выявление ошибок, предоставление обратной связи для итераций.
*   **Общение с пользователем:** Основная точка контакта с пользователем.
*   **Комплексные задачи:** Выполнение задач, требующих обширного контекста или глубоких рассуждений.

### 1.2. Junior Agent / Специалист по Выполнению (Qwen 0.5b через Ollama + Needle)

**Зона ответственности Qwen (LLM-движка Junior Agent):**
*   **Выполнение специализированных подзадач:** Фокусировка на делегированных кодоцентричных задачах.
*   **Генерация кода и контента:** Самостоятельное написание кода, функций, скриптов, генерация файлов конфигурации (`requirements.txt` и т.д.).
*   **Формулирование намерений инструментов:** Принятие решения о том, какой инструмент и с какими аргументами использовать.

## 2. Роль Needle: Интерпретатор Вызовов Инструментов

**Needle** выступает в качестве **тонкого слоя толколдинга/форматирования** для Qwen.
*   Получает *намерение* использовать инструмент от Qwen.
*   Преобразует это намерение в синтаксически корректный JSON-объект вызова инструмента.

## 3. Роль Pi: Среда Исполнения

**Pi** выполняет реальные инструменты (`default_api`) на основе инструкций Senior Agent, которые были сформированы после обработки предсказаний Needle.
